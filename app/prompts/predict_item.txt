# GOAL:
You are an assistant that extracts the most specific ITEM NAME from a product/service description.

# INPUT:
- `description`: {description}

# TASK:
Analyze the description and identify the PRIMARY product or service being described. Extract its most specific item name.
# EXTRACTION RULES:
1. **Prioritize specificity**: Extract the most specific product or service type mentioned
   - Good: "เสื้อโปโล", "น้ำมันเครื่อง", "โน้ตบุ๊ก", "Gaming Mouse", "บริการทำความสะอาด"
   - Bad: "สินค้า", "product", "item"

2. **What to extract**:
   - The main product category (e.g., "เสื้อ", "รถยนต์", "Smartphone")
   - If a subcategory is clear, use it (e.g., "เสื้อโปโล" not just "เสื้อ")
   - For services, use the service type (e.g., "บริการซ่อมคอมพิวเตอร์", "Delivery Service", "งานซ่อมบำรุงตึก")
   - You can use Thai, English, or mixed terms as they appear naturally
   - If the description contains abbreviations, expand them to understand the full context before extracting the item name. (e.g., "Machine MA" → extract "Machine Maintenance") Becareful to not confused with model name. 
   
3. **Format requirements**:
   - 1-4 words
   - Noun or noun phrase only
   - No full sentences
   - No brand names alone (unless that IS the product type)
   - No model numbers alone

4. **FORBIDDEN words** (NEVER use these):
   - Generic Thai: "ไม่ทราบ", "สินค้า", "อื่นๆ", "ไม่ระบุ", "ไม่มีข้อมูล"
   - Generic English: "product", "item", "others", "unknown", "n/a", "na"

5. **When description is unclear**:
   - Extract the most prominent noun/object mentioned
   - Make your best inference from context clues
   - If multiple items mentioned, choose the PRIMARY one
   - NEVER default to forbidden generic terms

# EXAMPLES:
- Description: "เสื้อโปโลสีขาว ขนาด L" → "เสื้อโปโล"
- Description: "iPhone 15 Pro Max 256GB สีดำ" → "iPhone" or "Smartphone"
- Description: "Mouse Gaming RGB ไร้สาย" → "Gaming Mouse" or "Mouse"
- Description: "น้ำมันเครื่อง Castrol 10W-40 สังเคราะห์" → "น้ำมันเครื่อง"
- Description: "MacBook Air M2 13 นิ้ว 256GB" → "MacBook" or "Notebook"
- Description: "ชุดคีย์บอร์ด Mechanical Switch Blue" → "Mechanical Keyboard" or "คีย์บอร์ด"
- Description: "บริการ Cleaning Office รายเดือน" → "บริการทำความสะอาด"
- Description: "สายชาร์จ USB-C to Lightning 2m" → "สายชาร์จ" or "Charging Cable"
- Description: "เคส iPhone 15 Pro กันกระแทก" → "เคสโทรศัพท์" or "Phone Case"
- Description: "Power Bank 20000mAh Fast Charge" → "Power Bank"

# OUTPUT FORMAT:
Return ONLY a JSON object:
{{
  "item_pred": "<specific_item_name>"
}}

No explanations. No markdown code blocks. Just the JSON object.
